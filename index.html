How Caesar Cipher Works

The Caesar cipher shifts each letter in the plaintext by a fixed number of positions down the alphabet. 


Encryption: C = (P + K) mod 26
Decryption: P = (C - K) mod 26

Where:
C = cipher text
P = plain text
K = shift key
